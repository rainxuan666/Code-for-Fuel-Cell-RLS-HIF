# Code-for-Fuel-Cell-RLS-HIF
This is origin code for the research articles "Excitation Signal-free Online Internal Parameter Identification for Proton Exchange Membrane Fuel Cell Diagnosis"
The source code for the DRT analysis module is derived from the research conducted by Ting Hei WAN, Francesco Ciucci, et al. The relevant code can be accessed via the team’s GitHub repository: https://github.com/ciuccislab. This project provides a compressed archive of the DRTtools package.
For the detailed parameter calculation and derivation processes, please refer to the files Formula derivation1.nb and Formula derivation2.nb.
The implementation based on the RLS-HIF algorithm is located within the fc_id directory.
The fault diagnosis and classification algorithms are detailed in the Fault diagnosis directory.
